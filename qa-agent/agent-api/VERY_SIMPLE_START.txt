========================================
HOW TO START THE SERVER
========================================

Copy and paste this ONE command into your Terminal:

cd /Users/amitkumarnigam/Downloads/openlens/qa-agent/agent-api && python3 SIMPLE_START.py

That's it! You will see the server output.

OR if that doesn't work, try:

cd /Users/amitkumarnigam/Downloads/openlens/qa-agent/agent-api
python3 -m pip install uvicorn fastapi
python3 -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8080

========================================
WHAT YOU'LL SEE
========================================

INFO:     Uvicorn running on http://0.0.0.0:8080
INFO:     UI served from: /Users/.../ui
INFO:     Application startup complete.

Then open: http://localhost:8080/ui/
